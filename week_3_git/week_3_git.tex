\section{Git}

\subsection{What is git}

\noindent
一个分布式的版本控制系统。分布式代表不需要联网，版本控制代表可以回溯文件修改历史。\\
有趣的部分：git是自托管的，也就是说git他自己的代码就是放在git仓库里的。现在你甚至可以在github上看到git的源代码。实现是用1000多行代码完成的。\\
git自带的git bash是一个命令行工具，可以用来操作git。他也很有用。\\
